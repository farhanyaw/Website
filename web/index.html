Apache

Installasi -> Config

jadi disini saya akan menjelaskan sedikit langkah langkah mengenai Apache, dari Install hingga configurasi web. ikuti saja langkah langkah dan foto dibawah ini

  • apt-get install apache2 (install)
  • systemctl status apache2 (cek status dan pastikan running)
  • nano /etc/network/interfaces (mengubah ip kalian, jangan lupa pastikan kalian sudah memakai host only)
  • systemctl restart networking (restart ip)
  • ip -c a (pastikan ip kalian sudah private/ip yang kalian tulis tadi)
  • nano /var/www/html/index.html (editlah untuk mengubah tampilan web browser kalian)
  • (pergi ke chrome atau browser kalian, lalu ketikkan ip private debian kalian tadi)

Installasi -> Config

jika kalian bertanya tanya "bang cara configurasi SSH gimana bang, bang cara pake SSH gimana bang" nih ikutin langkah langkah yang ada dibawah ini

  • apt-get install ssh (install)
  • systemctl status ssh (cek status ssh, pastikan running)
  • nano /etc/ssh/sshd_config
  • (kalian bisa bebah merubah port kalian, port digunakan untuk akses masuk nanti)
  • (jika sudah kalian bisa membuka putty atau cmd kalian di windows)

  • (kalian bisa menggunakan cmd dengan command)
  • user@ip -p port (ketikkan di cmd kalian)
